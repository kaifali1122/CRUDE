Getting Started with MongoDB

MongoDB is a popular NoSQL database that stores data in flexible, JSON-like documents. Here's a step-by-step guide to using MongoDB in your projects:

1. Installation

To get started with MongoDB:

  1. Download and install MongoDB Community Server from the official website
  2. Install MongoDB Compass (GUI tool) for easier database management
  3. Add MongoDB to your system PATH

2. Setting Up Your Project

To use MongoDB in your Node.js project:

  1. Install the required dependencies: 
    npm install mongoose express dotenv
  2. Create a `.env` file for your MongoDB connection string: 
    MONGODB_URI=mongodb://localhost:27017/your_database_name

3. Connecting to MongoDB

Here's how to connect to MongoDB in your Node.js application:

const mongoose = require('mongoose');
require('dotenv').config();

mongoose.connect(process.env.MONGODB_URI, {
    useNewUrlParser: true,
    useUnifiedTopology: true
})
.then(() => console.log('Connected to MongoDB'))
.catch(err => console.error('MongoDB connection error:', err));

4. Creating a Schema

Define your data structure using Mongoose schemas:

const studentSchema = new mongoose.Schema({
    name: { type: String, required: true },
    age: { type: Number, required: true },
    grade: { type: String, required: true },
    subjects: [String]
});

const Student = mongoose.model('Student', studentSchema);

5. CRUD Operations

Create (POST)

// Create a new student
const newStudent = new Student({
    name: "John Doe",
    age: 20,
    grade: "A",
    subjects: ["Math", "Science"]
});

await newStudent.save();

Read (GET)

// Get all students
const students = await Student.find();

// Get a specific student
const student = await Student.findById(id);

// Find with conditions
const gradeAStudents = await Student.find({ grade: "A" });

Update (PUT)

// Update a student
await Student.findByIdAndUpdate(id, {
    name: "New Name",
    age: 21
});

Delete (DELETE)

// Delete a student
await Student.findByIdAndDelete(id);
