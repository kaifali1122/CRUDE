MongoDB CRUD Explained

What is CRUD?

CRUD stands for Create, Read, Update, and Delete - the four fundamental operations of persistent storage. These operations form the foundation of database interactions in MongoDB.

Note: MongoDB is a NoSQL database that stores data in flexible, JSON-like documents.

Create Operations

MongoDB provides several ways to create documents:

Single Document Creation: 
// Insert one document
db.students.insertOne({
    name: "John Doe",
    age: 20,
    grade: "A",
    subjects: ["Math", "Science"]
})
Multiple Documents Creation: 
// Insert multiple documents
db.students.insertMany([
    {
        name: "Alice Smith",
        age: 21,
        grade: "B",
        subjects: ["History", "English"]
    },
    {
        name: "Bob Johnson",
        age: 19,
        grade: "A",
        subjects: ["Physics", "Chemistry"]
    }
])

Read Operations

MongoDB provides powerful querying capabilities:

Basic Queries: 
// Find all documents
db.students.find()

// Find with conditions
db.students.find({ grade: "A" })

// Find with multiple conditions
db.students.find({
    grade: "A",
    age: { $gt: 20 }
})
Advanced Queries: 
// Projection (select specific fields)
db.students.find({}, { name: 1, grade: 1 })

// Sorting
db.students.find().sort({ age: 1 })

// Limiting and Skipping
db.students.find().limit(5).skip(10)

// Aggregation
db.students.aggregate([
    { $match: { grade: "A" } },
    { $group: { _id: "$grade", count: { $sum: 1 } } }
])

Update Operations

MongoDB provides various update operations:

Single Document Update: 
// Update one document
db.students.updateOne(
    { name: "John Doe" },
    { $set: { age: 21 } }
)
Multiple Documents Update: 
// Update multiple documents
db.students.updateMany(
    { grade: "A" },
    { $set: { status: "Honor Roll" } }
)
Advanced Update Operations: 
// Using update operators
db.students.updateOne(
    { name: "John Doe" },
    {
        $set: { grade: "A+" },
        $inc: { age: 1 },
        $push: { subjects: "Computer Science" }
    }
)

// Upsert (update or insert)
db.students.updateOne(
    { name: "New Student" },
    {
        $set: { age: 18, grade: "B" }
    },
    { upsert: true }
)

Delete Operations

MongoDB provides different delete operations:

Single Document Delete: 
// Delete one document
db.students.deleteOne({ name: "John Doe" })
Multiple Documents Delete: 
// Delete multiple documents
db.students.deleteMany({ grade: "F" })

  • Implement proper error handling and validation
